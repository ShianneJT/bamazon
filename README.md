# Bamazon

**Bamazon** is a mock CLI storefront app.

## Setup

- **Node.js** is required

1. Clone the **Bamazon** repo to your computer.
2. Run the following command prompt in **Bamazon's** root folder:  
``` npm install ```

To begin, type ``` node bamazonCustomer.js ``` and follow the prompts.
